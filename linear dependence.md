# linear dependence
From [[linear algebra]]

## Definition
A list of vectors $v_{1}, \dots, v_{n}$ from $F$-vector space $V$ is linearly dependent iff $\exists a_{1}, \dots, a_{n} \in F, \exists a_{i} \ne 0: \sum\limits a_{i}v_{i} = 0$. Othwerise it is linearly independent.
