# premeasure
From [[measure theory]]
$\physics$
## Definition
Consider $S \subset 2^{X}$ and a set function $\mu: S \to [0, \infty]$ which is:
1. Countably monotone: $A \subset \cup B_{i} \implies \mu(A) \leq \sum\limits \mu(B_{i})$
2. “Preserves zero”: $\mu(\varnothing) = 0$
