# isometries have orthonormal matrices
From [[isometry]]
$\physics$
## Statement
If $T \in \L(V)$ is an [[isometry]], the columns of $[T]_{e_{[n]}}$ form an [[orthonormal basis]] for any [[orthonormal basis]] $e$.
Or without matrices: $Se_{1}, Se_{2}, \dots, Se_{n}$ form an [[orthonormal basis]].

## Proof
We know $\ev{e_{i}, e_{j}} = [i = j]$ and we know $\ev{Sv, Su} = \ev{v, u}$, so $\ev{S e_{i}, S e_{j}} = [i = j]$.
